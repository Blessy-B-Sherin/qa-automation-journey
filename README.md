# QA Automation Journey

This repository showcases my approach to building and maintaining a scalable QA automation framework using Java, Selenium, API testing, and CI practices.

## What this repository contains
- A production-style automation framework built from scratch
- API and UI test examples demonstrating end-to-end validation
- Supporting exercises and notes that explain design decisions and debugging approaches

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- REST Assured
- SQL
- Git & CI

## Focus
- Framework design and stability
- Automation ownership and debugging
- Practical test strategy, not just scripts
